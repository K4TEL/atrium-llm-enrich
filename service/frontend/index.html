API usage 📡

This page is a thin client over the REST API. The same calls from the command line:

curl -X POST "http://localhost:8000/extract_keywords" \
  -F "file=@lines.txt" -F "backend=openrouter" -F "top_k=10"

curl -X POST "http://localhost:8000/extract_keywords_text" \
  -H "Content-Type: application/json" \
  -d '{"lines": ["Výzkum odhalil základy kostela."], "top_k": 5}'
