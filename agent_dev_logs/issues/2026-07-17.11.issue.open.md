|----|----|
| PDF | Max visual and textual |
| InDesign XML | Font types, letter spacing, graphics regions, complex layout and tables, etc. | 
| HTML+CSS | Graphics + formatted text + complex layouts of tables and pages |
| DOCX | Same as above |
| Page/ALTO XML | Text regions and word-level positioning + graphics regions + hypnems for split words - table borders - actual visuals |
| MD | Simple tables and graphics regions + formatted text and headers + lists |
| TXT | Plain text ordered |
